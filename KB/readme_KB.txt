En esta carpeta se encuentran las bases de conocimiento para ambas categorías de ejercicios.

El archivo "basico1.dl" contiene el conjunto de reglas lógicas para clasificar y asignar los ejercicios a estudiantes con diferentes niveles de habilidad para la categoría de problemas tipo 1.
El archivo "basico11.dl" contiene las bases de conocimiento para la categoría de problemas 11.

El archivo "perfilAlum.dl" contiene el perfil del alumno.
El archivo "caracePr1.dl" contiene las características del problema.

Nota:  "caracePr1.dl" es un archivo que tomó el generador aleatorio para obtener estas características. Es un archivo con muchos ejemplos y cararacterísticas.
