En esta carpeta se guarda el historial de las corridas del sistema.

El archivo 'hist.ini' sólo contiene una línea para iniciar el archivo del historial.

'his.txt' contiene toda la información que genera el sistema, la intención de esto es usarla en un futuro para minería de datos.  Aquí se pueden consultar las preguntas asignadas, las respuestas correctas, los aciertos de cada estudiante, etc.
